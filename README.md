## Installation

1. Clone the project

```bash
git clone git@github.com:Bapugouda-B/react-mui-project.git
```
2. Navigate to the project directory

```bash
cd your-project-name
```

3. Install dependencies
```bash
npm install
```

4. Start the Server 
```bash
npm start
```
